# Episode 7-8. 오픈소스로 돈 버는 회사들의 비밀

"오픈소스는 공짜인데 어떻게 돈을 벌지?" 궁금하셨던 분들을 위한 비즈니스 모델 분석!

## 💼 오픈소스 비즈니스 모델들

### 1. 프리미엄 모델 (Freemium)
**무료 버전 + 유료 고급 기능**

- **MongoDB**: 기본 DB는 무료, Atlas(클라우드)는 유료
- **GitLab**: 커뮤니티 에디션 무료, 엔터프라이즈 유료
- **Slack**: 기본 기능 무료, 고급 기능 유료

### 2. 지원 서비스 모델
**Red Hat의 성공 공식**

- 오픈소스 Linux 배포판은 무료
- 기업용 지원, 컨설팅, 교육은 유료
- 2019년 IBM이 340억 달러에 인수!

<callout>
🏥 **병원으로 비유하면**
약(오픈소스)은 누구나 구할 수 있지만, 전문의 진료(지원 서비스)는 돈을 내야 합니다.
</callout>

### 3. 듀얼 라이센스 모델
**MySQL의 전략**

- **GPL 라이센스**: 무료 사용, 단 소스코드 공개 의무
- **상용 라이센스**: 유료 구매, 소스코드 비공개 가능

### 4. SaaS 모델
**클라우드로 수익화**

- **Elastic**: Elasticsearch 무료, Elastic Cloud 유료
- **MongoDB**: 데이터베이스 무료, Atlas 클라우드 유료
- **GitHub**: Git은 무료, GitHub 서비스는 유료

## 🎯 성공하는 오픈소스 기업의 특징

1. **강력한 커뮤니티**: 활발한 개발자 생태계
2. **명확한 가치 제안**: 무료 vs 유료의 차이점 명확화
3. **엔터프라이즈 친화적**: 기업이 필요로 하는 기능 제공
