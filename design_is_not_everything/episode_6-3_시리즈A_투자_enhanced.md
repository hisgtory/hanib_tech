## Part 1: Series A의 이해

### 💰 투자 라운드 단계

```
스타트업 투자 단계:

1. Pre-Seed (1-5억)
시점: 아이디어 + 팀
투자자: 엔젤, 액셀러레이터
목적: MVP 개발
지분: 5-10%

2. Seed (5-20억)
시점: MVP + 초기 고객
투자자: 시드 VC
목적: PMF 찾기
지분: 10-20%

3. Series A (30-100억) ← 여기!
시점: PMF + 성장 신호
투자자: Series A VC
목적: 스케일업
지분: 15-25%

4. Series B (100-300억)
시점: 검증된 성장
투자자: Growth VC
목적: 시장 확대
지분: 10-20%

5. Series C+ (300억+)
시점: 시장 리더
투자자: PE, 대형 VC
목적: 해외 진출, M&A
지분: 5-15%

각 단계별 기대치:
Seed: 가능성
Series A: 증명
Series B: 성장
Series C: 지배
```

### 📊 Series A 투자 기준

```
VC가 보는 핵심 지표:

1. Product-Market Fit
- MRR: $50K-100K+
- 성장률: MoM 15-20%+
- Retention: 6개월 80%+
- NPS: 40+

2. Unit Economics
- LTV/CAC > 3
- Payback < 12개월
- Gross Margin > 70%
- Burn Multiple < 2

3. 시장 규모
- TAM > $10B
- 연 성장률 > 20%
- 획득 가능 점유율 > 1%

4. 팀
- 도메인 전문성
- 실행력 증명
- 핵심 인재 보유
- 공동창업자 케미

5. 경쟁 우위
- 차별화된 제품
- 네트워크 효과
- 진입 장벽
- 특허/기술

실제 체크리스트:
□ 월 매출 1억+
□ 고객 100개+
□ 팀 10명+
□ 18개월 계획
□ 명확한 사용처
```

## Part 3: 밸류에이션과 조건

### 💵 기업 가치 산정

```
밸류에이션 방법론:

1. 배수 방식 (Revenue Multiple)
ARR × 배수 = 기업가치

SaaS 기준:
성장률 100%+: 10-20x
성장률 50-100%: 6-10x
성장률 30-50%: 3-6x

예시:
ARR $2M × 10 = $20M (240억)

2. 비교 기업 분석
유사 기업의 투자 사례
같은 시장, 비슷한 단계
지역별 차이 고려

3. VC 방식
필요 자금 ÷ 희석률 = 기업가치

예: 50억 필요, 20% 희석
50억 ÷ 0.2 = 250억 post

4. 미래 가치 할인
3년 후 Exit 가치 예상
→ 현재 가치로 할인
→ VC 목표 수익률 (10x)

Pre vs Post Money:
Pre-money: 투자 전 가치
Post-money: 투자 후 가치
Post = Pre + 투자금

예시:
Pre: 200억
투자: 50억
Post: 250억
희석: 50/250 = 20%
```

### 📜 Term Sheet 이해

```
주요 조항 해설:

경제적 조항:

1. Liquidation Preference
"1x non-participating"
= 투자금 먼저 회수 후 지분율대로

2. Anti-dilution
"Broad-based weighted average"
= 다운라운드 시 보호 (완화된 방식)

3. Dividend
"Non-cumulative 8%"
= 배당 시에만 8% (누적 안됨)

통제 조항:

1. Board Composition
"5명: 창업자 2, 투자자 2, 독립 1"
= 균형잡힌 이사회

2. Protective Provisions
투자자 동의 필요 사항:
- M&A
- 추가 투자
- 중요 자산 매각
- 사업 변경

3. Information Rights
- 월간 재무 리포트
- 연간 예산 승인
- 이사회 참관권

4. Vesting
"4년, 1년 cliff"
= 1년 후부터 3년간 분할

레드 플래그:
🚩 2x+ liquidation preference
🚩 Full ratchet anti-dilution
🚩 과도한 board control
🚩 Super pro-rata rights
🚩 짧은 vesting 기간
```
