## Part 6: 문화 차이 이해하고 극복하기

### 🌏 국가별 협업 스타일

```
🇺🇸 미국
- 직접적 피드백
- 개인 성과 중시
- "Great job!" 자주 사용
- 이메일 짧고 간결

🇯🇵 일본
- 간접적 표현
- 컨센서스 중요
- 침묵 = 동의 아님
- 디테일 중시

🇩🇪 독일
- 시간 엄수 필수
- 프로세스 중시
- 직설적 피드백
- 문서화 철저

🇮🇳 인도
- 관계 구축 우선
- 계층 구조 존중
- "Yes" ≠ 동의
- 컨텍스트 중요

🇧🇷 브라질
- 관계 중심
- 유연한 시간 개념
- 열정적 소통
- 개인적 대화 선호
```

### 💬 크로스컬처 커뮤니케이션

```
문화적 오해 방지 가이드:

상황 1: 피드백 차이
미국: "This needs to be completely redone"
한국 해석: "전체 실패작"
실제 의미: "몇 가지 수정 필요"
→ 해결: 구체적 수정사항 요청

상황 2: 데드라인 해석
독일: "금요일까지" = 금요일 09:00
한국: "금요일까지" = 금요일 23:59
인도: "금요일까지" = 다음주 월요일도 OK
→ 해결: 정확한 시간대 명시

상황 3: 동의 표현
일본: "검토하겠습니다" = 아마 NO
미국: "Let me think" = 아마 NO
한국: "긍정적으로 검토" = 아마 NO
→ 해결: 명확한 Yes/No 확인
```
