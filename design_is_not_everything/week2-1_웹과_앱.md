# 웹과 앱 (완성)

웹이랑 앱이 뭐가 다른가요? 같은 거 아니에요?

와인과 맥주처럼 비슷해 보이지만 다른 세계죠!

## #1 접근 방법이 달라요

**웹**: 브라우저에 URL 입력하면 끝

**앱**: 스토어에서 다운로드 → 설치 → 실행

이게 별거 아닌 것 같아도 사용자 경험에 큰 차이를 만들어요.

<hl>여기서 중요한 건, 웹은 URL로 중간부터도 접근 가능하다는 점을 고려해야 하고, 앱은 뒤로갔다가 돌아오는 등에 대해 고려해야 해요.</hl>

웹은 30초 안에 진입 장벽이 없지만, 앱은 설치라는 허들이 있죠.

## #2 업데이트 방식도 달라요

**웹**: 개발자가 서버에 올리면 즉시 적용

새로고침 하면 짠! 최신 버전.

**앱**: 스토어 심사 → 승인 → 사용자 업데이트

보통 1-2주 걸려요. 급한 버그 수정? 기다려야 해요.

<hl>이때문에 앱에서는 사용자가 업데이트를 안 할 가능성도 고려해야 하고, 이전 버전과의 호환성도 더 신중하게 고려해야 해요.</hl>
