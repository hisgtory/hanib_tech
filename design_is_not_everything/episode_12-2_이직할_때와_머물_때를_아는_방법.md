## 🧩 2단계: 단서 분석 - 머물 신호 확인

### 반증 A: 숨겨진 기회 발견
```
🔍 세밀 관찰:
- 새로운 프로젝트 소문 (아직 공개되지 않음)
- 상사 교체 예정 (인사팀 내부 정보)
- 회사 성장세 (매출 전년 대비 20% 증가)
```

### 반증 B: 이직 시장 현실 체크
```
📈 시장 조사 결과:
현재 연봉: 4500만원
이직 예상: 4200만원 (10% 하락 위험)
추가 고려사항: 
- 새 회사 적응 기간 6개월
- 연차 초기화
- 복리후생 격차
```

## 🎯 3단계: 종합 추리

### 추리 1: 감정 vs 이성의 대결
```
💭 탐정의 내면 분석:
감정적 판단: "여기 있으면 죽을 것 같아!"
이성적 판단: "객관적 데이터를 보면..."

결론: 감정이 앞설 때는 2주 대기 원칙 적용
```

### 추리 2: 타이밍 수사
```
⏰ 최적 이직 시기 추정:
Q1: 신입 채용 시즌 (경쟁 치열)
Q2: 중간 채용 활발 ✅ 최적
Q3: 하반기 예산 집행
Q4: 인사 정리 시기 (위험)
```

```

☐ 연봉협상 전략 수립
☐ 퇴사 시나리오 작성
```

## 🎭 최종 결론: 탐정의 조언

### 🔍 이직을 결정해야 할 때
1. **성장 정체가 1년 이상 지속**될 때
2. **회사의 미래가 불투명**할 때
3. **가치관 충돌이 해결 불가능**할 때
4. **더 나은 기회가 확실**할 때

### 🏠 머물러야 할 때
1. **새로운 도전 기회가 예정**되어 있을 때
2. **시장 상황이 불리**할 때
3. **개인적 변화(결혼, 육아 등)가 있을** 때
4. **감정적으로만 떠나고 싶을** 때
