# Episode 6-4: 스톡옵션, 종이 위의 부자가 진짜 부자가 되려면

## 🎬 Scene: 충격적인 세금 고지서

```
2021년, 유니콘 스타트업 디자이너

"축하해요! 회사 가치 1조원 돌파!"
디자이너: "제 스톡옵션 가치가 50억이네요!"

[행사 결정]
행사가: 1만원
현재가: 100만원
수량: 5,000주
장부상 이익: 49.5억

[3개월 후]
세금 고지서: 15억원
디자이너: "뭐?? 주식 팔지도 않았는데?"

[6개월 후]
회사 가치 폭락, 주가 10만원
실제 가치: 5억
낸 세금: 15억

"종이 부자에서 진짜 빚쟁이가 되었어요..."
```

**스톡옵션은 복권이 아니라 복잡한 금융상품입니다.**

## Part 1: 스톡옵션 기초

### 📊 스톡옵션이란?

```
Stock Option = 주식을 살 수 있는 권리

기본 구조:
회사: "1만원에 주식 살 권리 줄게"
직원: "나중에 주가 오르면 사서 팔게"

4가지 핵심 요소:
1. Grant (부여)
   입사 시 계약
   수량과 조건 결정

2. Vesting (베스팅)
   권리 획득 기간
   보통 4년

3. Exercise (행사)
   실제로 주식 구매
   행사가격 지불

4. Exit (청산)
   IPO or M&A
   실제 현금화

예시:
부여: 10,000주 @ 10,000원
4년 후: 주가 100,000원
행사 시: (100,000 - 10,000) × 10,000 = 9억 이익
```

### ⏰ Vesting Schedule

```
일반적인 베스팅:

4-year vesting with 1-year cliff

Year 0-1: 0% (Cliff)
Year 1: 25% (2,500주)
Year 1-2: 매월 2.08% (월 208주)
Year 2: 누적 50% (5,000주)
Year 3: 누적 75% (7,500주)
Year 4: 100% (10,000주)

특수 조건:

1. Accelerated Vesting
M&A 시 즉시 100% 베스팅
"Single Trigger" vs "Double Trigger"

2. Early Exercise
베스팅 전 미리 행사
세금 혜택 가능
리스크 존재

3. Clawback
특정 조건 미달 시 회수
성과 미달, 경쟁사 이직

스타트업 단계별 베스팅:
초기 (Seed): 3-4년
성장기 (Series A-B): 4년
후기 (Series C+): 4-5년
```

## Part 2: 세금의 함정

### 💸 행사 시점 세금

```
한국 세금 체계:

1. 행사 시점 (근로소득세)
과세표준 = (시가 - 행사가) × 주식수
세율: 최대 45% + 지방세

예시:
시가: 100만원
행사가: 1만원
수량: 1,000주
과세표준: 9.9억
세금: 약 4억

2. 매도 시점 (양도소득세)
대주주: 20-25%
소액주주: 비과세 or 10%

미국 세금 체계:

1. NSO (Non-Qualified)
행사 시: 일반 소득세
매도 시: 자본이득세

2. ISO (Incentive)
행사 시: AMT만
매도 시: 자본이득세
1년 홀딩 시 혜택

절세 전략:
✅ 조기 행사 (낮은 가치 때)
✅ 분할 행사 (세율 구간 관리)
✅ RSU 고려 (세금 회사 부담)
❌ 고점 행사 (세금 폭탄)
```

### 📉 리스크 시나리오

```
최악의 시나리오들:

Case 1: 세금 > 실제 이익
행사가: 1만원
행사 시 주가: 100만원
세금 납부: 40억
현재 주가: 10만원
실제 가치: 4억
순손실: -36억

Case 2: 회사 망함
스톡옵션 가치: 0
낸 세금: 환급 불가
투자한 행사 자금: 손실

Case 3: Lock-up 기간
IPO 후 6개월 매도 금지
그 사이 주가 폭락
세금은 이미 납부

Case 4: 환율 리스크
해외 상장 회사
원화 강세 시 손실
이중 과세 위험

보호 방법:
1. 회사 실적 지속 모니터링
2. 행사 시점 신중히 결정
3. 세금 자금 미리 준비
4. 부분 행사 전략
5. 세무사 상담 필수
```

## Part 3: 협상과 평가

### 💼 스톡옵션 협상

```
입사 제안 평가:

제안: "연봉 8천 + 스톡옵션 10,000주"

체크 포인트:
1. 현재 회사 가치? → 100억
2. 총 발행 주식수? → 1,000만주
3. 내 지분율? → 0.1%
4. 행사가격? → 현재 FMV
5. 베스팅 조건? → 4년
6. 조기 행사 가능? → 불가

가치 계산:
현재 가치: 10,000 × 1만원 = 1억
목표 가치 (10x): 10억
연간 기대 가치: 2.5억

협상 포인트:
"Series A 직전이니 더 많은 옵션을..."
"연봉 낮춘 만큼 옵션으로..."
"조기 행사 권한 주세요"
"Refresh grant 약속"

직급별 일반적 지분:
첫 엔지니어: 0.5-1%
첫 디자이너: 0.25-0.5%
10번째 직원: 0.1-0.25%
50번째 직원: 0.01-0.05%
```

### 📈 Exit 시나리오

```
Exit별 수익 계산:

시나리오 1: IPO
회사 가치: 1조원 (100x)
내 지분: 0.1% = 10억
세금: 4억
순수익: 6억

시나리오 2: M&A
인수가: 5,000억 (50x)
내 지분: 0.1% = 5억
세금: 2억
순수익: 3억

시나리오 3: Secondary
VC가 지분 매입
할인율: 20-30%
즉시 현금화 가능
미래 상승 포기

시나리오 4: 망함
가치: 0
스톡옵션: 휴지
경력: 값진 경험

확률적 기대값:
IPO (5%): 6억 × 0.05 = 0.3억
M&A (15%): 3억 × 0.15 = 0.45억
Secondary (10%): 1억 × 0.1 = 0.1억
실패 (70%): 0 × 0.7 = 0
기대값: 0.85억/년
```

## Part 4: 스톡옵션 관리

### 📋 행사 전략

```
행사 시점 결정:

1. 83(b) Election (미국)
조기 행사 + 세금 신고
미래 차익에 대한 세금 절감

2. 한국 조기 행사
저가 행사로 세금 최소화
리스크: 회사 망하면 손실

3. 분할 행사
매년 일부씩 행사
세율 구간 관리
리스크 분산

4. Exit 시점 행사
확실한 수익 시점
최대 세금 부담
안전한 선택

의사결정 트리:
회사 전망 좋음?
├─ YES → 자금 있음?
│  ├─ YES → 조기 행사
│  └─ NO → 대기
└─ NO → 행사 안함

행사 자금 마련:
1. 개인 자금
2. 대출 (주식 담보)
3. Cashless Exercise
4. 회사 지원 (드물게)
```

### 🏦 포트폴리오 관리

```
스톡옵션 리스크 관리:

집중 리스크:
전재산의 90% = 한 회사 스톡
→ 위험한 베팅

분산 전략:
1. 일부 Secondary 매도
2. 다른 자산 투자
3. 여러 스타트업 경험

경력 플랜:
Stage 1: 초기 스타트업 (높은 지분)
Stage 2: 성장 스타트업 (중간 지분)
Stage 3: 후기 스타트업 (낮은 지분, 높은 확률)

스톡옵션 관리 도구:
- Carta: 지분 관리
- Pulley: 가치 계산
- Excel: 시나리오 분석

기록 관리:
□ Grant Letter 보관
□ 행사 가격 기록
□ 베스팅 일정 캘린더
□ 회사 가치 추적
□ 세무 서류 보관
```

## Part 5: RSU와 대안

### 🎁 RSU vs Stock Option

```
RSU (Restricted Stock Unit):

장점:
- 무료로 주식 받음
- 하방 리스크 없음
- 세금 회사가 일부 부담

단점:
- 상승 여력 제한
- 즉시 과세
- 대기업 위주

비교:
           Stock Option    RSU
리스크        높음         낮음
수익성        높음         중간
세금       행사 시      부여 시
초기비용      있음         없음
적합성    초기 스타트업  후기/상장사

Phantom Stock:
- 실제 주식 없이 차익만
- 현금 정산
- 세금 단순

ESPP (Employee Stock Purchase Plan):
- 할인가 주식 구매
- 보통 15% 할인
- 상장사 위주

Profit Sharing:
- 이익 배분
- 즉시 현금
- 지분 없음
```
