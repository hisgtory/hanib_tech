# 웹개발과 앱개발의 차이 (완성)

## 웹 개발

**기술 스택**: HTML, CSS, JavaScript, React 등
**장점**: 
- 크로스 플랫폼 (모든 기기에서 동작)
- 업데이트 용이
- 개발 비용 저렴

**단점**:
- 네이티브 기능 제한
- 성능 상대적으로 느림

## 앱 개발

**기술 스택**: 
- iOS: Swift, Objective-C
- Android: Kotlin, Java

**장점**:
- 높은 성능
- 네이티브 기능 활용
- 오프라인 사용 가능

**단점**:
- 플랫폼별 개발 필요
- 스토어 심사 필요
- 높은 개발 비용
