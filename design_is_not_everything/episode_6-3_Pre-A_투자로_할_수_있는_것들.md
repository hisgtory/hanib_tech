# Episode 6-3: Pre-A 투자로 할 수 있는 것들
## "5억이 생겼다! 이제 뭘 할 수 있을까?"

### 오프닝: 투자금 쇼핑 시작하기

"축하합니다! Pre-A 투자 5억을 유치했네요!"

그런데 잠깐, 이 돈으로 뭘 할 수 있을까요? 마치 대형마트에 처음 가본 아이처럼, 모든 게 다 갖고 싶고 다 필요해 보입니다. 하지만 장바구니(회사 통장)는 한정되어 있죠.

Pre-A 투자금은 보통 5-10억 규모입니다. 듣기엔 큰 돈 같지만, 스타트업에게는 생존과 성장을 위한 연료일 뿐이에요. 잘못 쓰면 금세 바닥나고, 제대로 쓰면 다음 라운드까지 버틸 수 있는 마법의 돈입니다.

오늘은 Pre-A 투자금으로 진짜 살 수 있는 것들을 쇼핑 카탈로그처럼 정리해드릴게요.

## Pre-A 투자금 쇼핑 리스트

### 🏪 인재마트 - "사람 사기"
**예산: 전체 투자금의 60-70%**

**🛒 개발자 1명 (시니어)**
- 가격: 연봉 8천만원 + 스톡옵션 + 4대보험
- 설명: 제품을 실제로 만들어줄 마법사. 없으면 아무것도 할 수 없음
- 재고: 항상 부족, 조기품절 주의
- 리뷰: "비싸지만 없으면 망함" ⭐⭐⭐⭐⭐

**🛒 개발자 2명 (주니어)**
- 가격: 각각 연봉 5천만원 + 스톡옵션
- 설명: 시니어가 시키는 일 열심히 해줄 착한 아이들
- 장점: 열정 넘침, 성장 가능성 높음
- 단점: 멘토링 필요

**🛒 마케터 1명**
- 가격: 연봉 6천만원
- 설명: 제품 만들어놨는데 아무도 모르면 안 되잖아요
- 필수조건: 퍼포먼스 마케팅 경험 2년 이상
- 주의사항: 브랜딩만 하는 마케터 말고 '숫자로 말하는' 마케터

**🛒 디자이너 0.5명 (프리랜서)**
- 가격: 월 300만원 (필요할 때만)
- 설명: 예쁘게 만들어주는 사람
- 팁: 풀타임 말고 프로젝트 단위로 쓰세요

- 투자효율: 높음 (개발속도 2배 향상)

## 필수템 vs 사치템 구분

### ✅ 필수템 (생존용품)
1. **개발자 최소 2명** - 제품 없으면 게임 끝
2. **사무공간** - 집에서 일하면 집중 안 됨
3. **기본 서버비** - 서비스 다운되면 신뢰도 추락
4. **최소한의 마케팅비** - 고객 없으면 매출 없음

### ❌ 사치템 (나중에 사도 되는 것들)
1. **강남 사무실** - 주소 좋다고 매출이 늘지 않음
2. **비싼 의자/책상** - 허먼밀러 의자는 IPO 후에
3. **회사 로고 브랜딩** - 제품이 먼저, 브랜딩은 나중에
4. **각종 컨설팅** - 돈 주고 조언 듣기보다 직접 부딪히기

## 돈 쓰는 순서와 타이밍

### 1단계: 생존 기반 구축 (투자금의 40%)
**첫 1개월**
- 핵심 개발자 채용
- 최소한의 사무공간 확보
- 기본 개발환경 셋팅

### 2단계: 제품 완성 (투자금의 30%)  
**2-4개월**
- MVP(최소기능제품) 개발 완료
- 베타 테스트 진행
- 초기 피드백 반영

### 3단계: 시장 검증 (투자금의 20%)
**5-8개월**
- 마케팅 시작
- 고객 획득 채널 테스트
- 제품-시장 적합성 확인

### 4단계: 성장 준비 (투자금의 10%)
**9-12개월**
- 다음 라운드 준비
- 팀 확장
- 시스템 고도화

## 망한 스타트업들의 돈 쓰기 패턴

### 😱 실패 케이스 1: "우리도 배민처럼"

- 결과: 브랜드 인지도는 높아졌지만 매출 0원
- 교훈: 제품 없이 브랜딩은 의미 없음

### 😱 실패 케이스 2: "개발자가 곧 실력"
**문제**: 시니어 개발자 5명 채용
- 월 인건비만 4천만원 (연 4.8억)
- 결과: 8개월 만에 돈 떨어짐
- 교훈: 적절한 시니어-주니어 비율 유지 필요

### 😱 실패 케이스 3: "일단 많이 뽑자"
**문제**: 각 팀별로 사람부터 채용
- 15명 팀으로 시작 (월 인건비 7천만원)
- 결과: 6개월 만에 자금 소진
- 교훈: 검증되지 않은 비즈니스 모델에 인력 투입은 위험

### 😱 실패 케이스 4: "강남에 있어야 간지"
**문제**: 강남역 30평 사무실 임차
- 보증금 5천만원 + 월세 800만원
- 결과: 고정비 부담으로 자금 압박
- 교훈: 초기에는 공유오피스로 충분

## ROI 관점에서 투자 효율성

### 🏆 ROI 높은 투자 TOP 5

1. **시니어 개발자 1명**: 투입 8천만원 → 제품 완성 → 무한 가치
2. **퍼포먼스 마케터**: 투입 6천만원 → 고객획득 시스템 구축 → 지속적 매출
3. **AWS 서버**: 투입 600만원 → 서비스 안정성 → 고객 신뢰도 증가  
4. **듀얼 모니터**: 투입 200만원 → 개발 생산성 2배 → 빠른 제품 출시
5. **A/B 테스트 도구**: 투입 50만원 → 전환율 30% 개선 → 매출 증대

### 📉 ROI 낮은 투자 TOP 5

1. **고급 사무실**: 투입 5천만원 → 간지 증가 → 매출 변화 없음
2. **브랜딩 컨설팅**: 투입 3천만원 → 예쁜 로고 → 고객 증가 미미
3. **각종 컨퍼런스 참가**: 투입 500만원 → 네트워킹 → 직접적 성과 미지수
4. **고급 의자/책상**: 투입 1천만원 → 직원 만족도 → 생산성 증대 미미  
5. **과도한 팀빌딩**: 투입 300만원 → 팀워크 → 단기 성과 없음

## 실전 팁: 돈 관리하기

### 💰 자금 관리 3원칙

1. **18개월 런웨이 유지**
   - 투자금은 18개월은 버텨야 함
   - 월 고정비를 투자금의 5.5%로 제한

2. **3:3:3:1 비율 법칙**
   - 인건비 30%, 마케팅 30%, 개발비 30%, 기타 10%

3. **매월 번레이트 체크**
   - 매달 얼마씩 쓰는지 정확히 추적
   - 6개월 후 재검토하여 조정

### 📊 필수 지표 관리

1. **CAC (Customer Acquisition Cost)**
   - 고객 한 명 획득하는데 드는 비용
   - 목표: LTV의 1/3 이하

2. **LTV (Life Time Value)**  
   - 고객 한 명이 평생 가져다주는 매출
   - 목표: CAC의 3배 이상

3. **번레이트 (Burn Rate)**
   - 매월 소모되는 자금
   - 목표: 예상보다 20% 이하로 유지

## 마무리: 현명한 투자금 사용법

Pre-A 투자금 5억은 생각보다 빨리 없어집니다. 특히 사람을 뽑기 시작하면 매월 몇천만원씩 나가죠.

**핵심은 우선순위입니다.**

1. 제품 만들 사람부터 (개발자)
2. 제품 팔 사람 다음에 (마케터)  
3. 제품 포장할 사람은 나중에 (디자이너)
4. 간지용 투자는 절대 하지 마세요
