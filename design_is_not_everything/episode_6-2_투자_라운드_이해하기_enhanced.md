## Part 1: 투자 라운드 완벽 이해

### 🌱 Seed Round (시드 라운드)

```
단계: 아이디어 → 초기 제품
금액: 1-5억원
투자자: 엔젤투자자, 초기 VC
기간: 6-12개월 런웨이

회사 상황:
- 직원 3-5명
- MVP 개발 중
- 매출 거의 없음
- 생존이 목표

디자이너 현실:
- 혼자 모든 디자인
- 일당백 멀티 플레이어
- 주 80시간 근무
- 스톡옵션이 유일한 희망

실제 대화:
"월급은 최저임금이지만 스톡옵션 드릴게요"
"언제 가치가 생기나요?"
"IPO 하면... 한 7년 후?"
```

```

- 초기 고객 확보
```

```
- 전문 영역 분화
- 프로세스 정립
- 채용 참여
```


## Part 3: 투자 유치 과정과 디자이너

```

   - 시장의 문제
   - 고객의 고통

2. Solution (3-4장)
   - 우리의 해결책
   - 제품 스크린샷 ⭐

   - TAM, SAM, SOM
   - 성장 가능성

4. Business Model (2장)
   - 수익 모델
   - 유닛 이코노믹스

5. Traction (3-4장)
   - 성장 그래프 ⭐
   - 핵심 지표

6. Team (2장)
   - 핵심 멤버
   - 경력과 성과

7. Ask (1장)
   - 투자 금액
   - 사용 계획

디자이너 기여도: 40%
(비주얼이 투자 결정의 30%를 좌우)
```

## Part 4: 밸류에이션과 지분 희석

### 💰 회사 가치 평가 이해하기

```
밸류에이션 계산:

Pre-money Valuation (투자 전 가치)
+ Investment (투자금)
= Post-money Valuation (투자 후 가치)

예시:
투자 전 가치: 100억
투자금: 20억
투자 후 가치: 120억
투자자 지분: 20/120 = 16.67%

당신의 지분 변화:
시드: 1% (1억 가치)
  ↓ 
시리즈A: 0.8% (8억 가치)
  ↓
시리즈B: 0.6% (30억 가치)
  ↓
시리즈C: 0.4% (80억 가치)

지분은 줄지만 가치는 증가!
```

### 📉 다운 라운드의 공포

```
정상 라운드:
시리즈A: 100억 → 시리즈B: 500억 ✅

다운 라운드:
시리즈A: 100억 → 시리즈B: 50억 ❌

다운 라운드 시나리오:
1. 전 직원 사기 저하
2. 스톡옵션 가치 하락
3. 핵심 인재 이탈
4. 추가 투자 어려움
5. 매각 또는 폐업

실제 사례:
"우리 옵션 행사가가 현재 주가보다 높아요"
= 스톡옵션 휴지조각
```

## Part 6: 투자 라운드별 생존 가이드

### 🎪 각 단계별 생존 전략

#### Seed 단계 생존법
```
DO:
✅ 다양한 스킬 습득
✅ 빠른 실행력
✅ 창업자와 친해지기
✅ 주인의식 갖기

DON'T:
❌ 완벽주의
❌ 프로세스 요구
❌ 워라밸 기대
❌ 단기 수익 기대

월급쟁이 마인드 = 즉시 퇴사
```

#### Series A 단계 전략
```
DO:
✅ 프로세스 정립
✅ 팀 빌딩 시작
✅ 디자인 시스템 구축
✅ 성과 지표 추적

DON'T:
❌ 혼자 다 하기
❌ 채용 대충하기
❌ 문서화 미루기
❌ 정치 시작하기
```

#### Series B+ 단계 정치
```
DO:
✅ 가시성 확보
✅ 네트워킹
✅ 성과 어필
✅ 리더십 개발

DON'T:
❌ 라인 잘못 타기
❌ 파벌 만들기
❌ 험담하기
❌ 안주하기
```
