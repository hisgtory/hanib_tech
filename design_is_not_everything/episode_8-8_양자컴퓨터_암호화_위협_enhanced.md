## Part 2: 암호화가 깨지는 원리

### 🔓 쇼어 알고리즘 (Shor's Algorithm)

```javascript
// RSA 암호화의 원리와 양자 공격

class RSAEncryption {
  // 현재 RSA 암호화 방식
  traditional() {
    // 1. 두 큰 소수 선택
    const p = 1234567890123456789n;  // 큰 소수 1
    const q = 9876543210987654321n;  // 큰 소수 2
    
    // 2. 곱셈 (쉬움)
    const n = p * q;  // 공개키의 일부
    
    // 3. 역연산 (어려움)
    // n을 보고 p와 q 찾기 = 매우 어려움
    
    return {
      공개키: n,
      비밀키: [p, q],
      
      클래식_해독시간: {
```

## 🎁 Bonus: 양자 컴퓨터 체험하기

### 🖥️ 직접 해보는 양자 프로그래밍

```python
# IBM Quantum으로 첫 양자 프로그램 작성

# 1. IBM Quantum 가입 (무료)
# https://quantum-computing.ibm.com/

# 2. Qiskit 설치
# pip install qiskit

from qiskit import QuantumCircuit, execute, Aer

# 양자 회로 만들기
def create_quantum_hello():
    # 2큐비트 회로 생성
    qc = QuantumCircuit(2, 2)
    
    # 첫 번째 큐비트를 중첩 상태로
    qc.h(0)  # Hadamard 게이트
    
    # 얽힘 생성
    qc.cx(0, 1)  # CNOT 게이트
    
    # 측정
    qc.measure([0, 1], [0, 1])
    
    return qc

# 시뮬레이터에서 실행
circuit = create_quantum_hello()
backend = Aer.get_backend('qasm_simulator')
job = execute(circuit, backend, shots=1000)
result = job.result()
counts = result.get_counts(circuit)

print(f"결과: {counts}")
# 출력: {'00': ~500, '11': ~500}
# 50% 확률로 00 또는 11 (얽힘 상태)
```
