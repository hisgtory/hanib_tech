# Episode 13-1: 스타트업 vs 대기업, 디자이너의 선택

## 🎬 Scene: 두 개의 오퍼레터 앞에서


**이 에피소드를 읽고 나면, 당신은 확실한 답을 가질 수 있을 것이다.**

## Part 2: 성장 곡선의 차이

### 📈 3년 후 모습 예측

#### 스타트업 3년 후

**Best Case (상위 10%)**
```
- 리드 디자이너 / 디자인 팀장
- 연봉 8,000만원
- 스톡옵션 가치 5억 (if 엑싯)
- 0→1 프로덕트 런칭 경험 다수
- 업계 인지도 상승
- 창업 or 다음 스타트업 C레벨
```

**Average Case (중위 50%)**
```
- 시니어 디자이너
- 연봉 6,500만원
- 스톡옵션 휴지조각
- 다양한 실무 경험
- 이직 시 대기업 입사 가능
```

**Worst Case (하위 10%)**
```
- 회사 망함
- 번아웃으로 퇴사
- 경력 공백
- 포트폴리오 부실 (보안 이슈)
```

#### 대기업 3년 후

**Best Case (상위 10%)**
```
- 시니어 디자이너
- 연봉 8,500만원
- 해외 지사 발령 기회
- 글로벌 프로젝트 경험
- MBA 지원
```

**Average Case (중위 50%)**
```
- 중급 디자이너
- 연봉 7,500만원
- 안정적 커리어
- 전문 분야 확립
- 워라밸 확보
```

**Worst Case (하위 10%)**
```
- 정체된 성장
- 조직 개편으로 부서 이동
- 매너리즘
- 이직 시 연봉 협상 어려움
```

## Part 3: 실무자들의 진짜 이야기

### 🗣 인터뷰: 스타트업 → 대기업 이동한 디자이너 A

> "스타트업에서 2년 일하고 네이버로 이직했어요.
> 
> **좋아진 점:**
> - 체계적인 프로세스 (더 이상 즉흥 디자인 없음)
> - 정시 퇴근 (가족과 시간 보내기 가능)
> - 안정적인 수입 (대출 받기도 쉬워짐)
> 
> **아쉬운 점:**
> - 의사결정 속도 (한 달 걸려서 버튼 색상 결정)
> - 임팩트 체감 (내 디자인이 묻히는 느낌)
> - 정치적 고려 (윗선 눈치 보기)
> 
> **결론:** 
> 20대 후반까지는 스타트업, 30대부터는 대기업 추천"

### 🗣 인터뷰: 대기업 → 스타트업 이동한 디자이너 B

> "삼성에서 5년 일하다가 시리즈A 스타트업으로 왔어요.
> 
> **좋아진 점:**
> - 빠른 실행 (아이디어가 바로 제품에)
> - 다양한 경험 (디자인, 마케팅, 채용까지)
> - 주인의식 (내가 만든 서비스)
> 
> **힘든 점:**
> - 불확실성 (내년에 회사가 있을까?)
> - 리소스 부족 (디자인 시스템? 직접 만들어야)
> - 연봉 삭감 (30% 다운, 스톡은 미지수)
> 
> **결론:**
> 안정성 포기하고 성장 선택. 후회는 없지만 쉽지 않음"

## Part 4: 면접 전략의 차이

### 🎯 스타트업 면접 준비

#### 포트폴리오 구성
```
스타트업이 원하는 것:
1. 문제 해결 과정 > 완성도
2. 빠른 iteration 능력
3. 0→1 경험
4. 사업 이해도
5. 실행력

포트폴리오 구조:
- 프로젝트 3개 (깊이 있게)
- 실패 사례와 학습
- 사이드 프로젝트 필수
- 코딩 능력 (플러스 요소)
```

#### 예상 질문과 답변
```
Q: "우리 서비스 써보셨나요? 개선점은?"

Q: "리소스가 부족한데 어떻게 일하시겠어요?"
A: "MVP 접근으로 핵심 기능부터... 과거 경험 예시..."

Q: "개발자가 불가능하다고 하면?"
A: "대안 3개 준비, 기술 제약 이해하고 협의..."
```

### 🎯 대기업 면접 준비

#### 포트폴리오 구성
```
대기업이 원하는 것:
1. 체계적 프로세스

4. 완성도
5. 글로벌 감각

포트폴리오 구조:
- 프로세스 중심 설명
- 다양한 이해관계자 협업
```

```
Q: "글로벌 사용자를 위한 디자인 경험은?"

Q: "디자인 시스템 구축 경험은?"

```

## Part 5: 현실적인 선택 가이드

### 🎭 당신에게 맞는 곳은?

#### 스타트업을 선택해야 할 때

✅ 이런 사람이라면:
- 리스크 감수 가능
- 빠른 성장 욕구
- 다양한 경험 원함
- 자율성 중요
- 창업 계획 있음

✅ 이런 상황이라면:
- 20대 ~ 30대 초반
- 경제적 부담 적음
- 번아웃 경험 없음
- 포트폴리오 빈약
- 전문성 탐색 중

#### 대기업을 선택해야 할 때

✅ 이런 사람이라면:
- 안정성 중요
- 체계적 성장 선호
- 워라밸 필수
- 전문성 심화 원함
- 글로벌 경험 원함

✅ 이런 상황이라면:
- 가족 부양 책임
- 번아웃 회복 필요
- 대출/육아 계획
- 이미 다양한 경험 보유
- 장기 커리어 플랜

### 💡 스마트한 커리어 패스

#### Plan A: 지그재그 전략
```
20대 중반: 스타트업 (2-3년)
         ↓ 다양한 경험, 빠른 성장
20대 후반: 중견기업 (2년)
         ↓ 체계 학습, 안정성 확보
30대 초반: 대기업 (3-4년)
         ↓ 전문성 확립, 네트워크
30대 중반: 스타트업 (C레벨) or 창업
```

#### Plan B: 깊이 우선 전략
```
신입: 대기업 입사
    ↓ 탄탄한 기초, 체계적 교육
3-5년차: 스타트업 이직
    ↓ 실무 경험 극대화
7-10년차: 대기업 복귀 (시니어)
    ↓ 높은 연봉, 안정성
10년차+: 독립 or 창업
```
