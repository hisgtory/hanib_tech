## 카드 1: 표지

**"내 컴퓨터에선 되는데요?"**

개발자가 이 말을 하는 진짜 이유

---

## 카드 2: 왜 이런 일이?

🤔 **환경이 다르면 결과도 다르다**

| 내 컴퓨터 | 다른 환경 |
|----------|----------|
| 최신 OS | 구버전 OS |
| 특정 브라우저 | 다른 브라우저 |
| 빠른 인터넷 | 느린 인터넷 |
| 테스트 데이터 | 실제 데이터 |

---

## 카드 3: 환경 차이 예시

**같은 코드, 다른 결과**

- 🍎 macOS에선 OK → 🪟 Windows에선 에러
- 🦊 Firefox에선 OK → 🧭 Safari에선 깨짐
- 📱 최신폰에선 OK → 📱 구형폰에선 느림
- 🏠 회사에선 OK → ☕ 카페에선 안됨

---

## 카드 4: 버그 리포트 필수 정보

📝 **이것만 알려주세요**

1. **뭘 했는지** - 어떤 버튼을 눌렀나요?
2. **뭐가 나왔는지** - 어떤 에러가 보였나요?
3. **어디서 했는지** - OS, 브라우저, 앱 버전
4. **스크린샷** - 백 마디 말보다 한 장의 사진

---

## 카드 5: 좋은 버그 리포트

✅ **이렇게 말해주세요**

> "iOS 17, Safari에서
> 결제 버튼을 누르면
> 로딩만 돌고 안 넘어가요"
> [스크린샷 첨부]

❌ "결제가 안 돼요"
