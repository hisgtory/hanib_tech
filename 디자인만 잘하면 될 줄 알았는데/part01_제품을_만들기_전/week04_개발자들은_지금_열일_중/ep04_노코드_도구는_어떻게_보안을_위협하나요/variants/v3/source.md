## 카드 2: Shadow IT란?

👻 **섀도우 IT**

회사가 승인하지 않은 IT 도구를
직원이 몰래 사용하는 것

예시:
- 개인 구글 드라이브에 회사 문서 저장
- 승인 없이 Notion에 고객 정보 정리
- 개인 카카오톡으로 업무 대화

---

## 카드 3: 노코드의 보안 위험

⚠️ **이런 위험이 있어요**

| 위험 | 설명 |
|------|------|
| 🔓 데이터 유출 | 외부 서버에 정보 저장 |
| 👥 접근 관리 | 퇴사자도 접근 가능 |
| 📋 규정 위반 | ISMS, 개인정보보호법 |
| 🔗 연동 위험 | 과도한 권한 부여 |

## 카드 4: 위험 신호

🚨 **이런 상황이면 주의!**

- 고객 개인정보를 노코드 도구에 저장
- "아무나" 볼 수 있는 공유 링크
- 퇴사자 계정이 그대로
- 어떤 데이터가 어디에 있는지 모름
