### 노코드 도구의 보안 위험

**1. 데이터가 어디로 가는지 모름**
```
회사 데이터 → Zapier → Google Sheets → 외부 서버?
데이터 흐름을 회사가 파악할 수 없어요
```

**2. 접근 권한 관리 어려움**
```
N님이 퇴사하면?
- 개인 계정으로 만든 자동화
- 회사 데이터에 여전히 접근 가능
- 누가 어떤 권한 있는지 파악 불가
```

**3. 컴플라이언스 문제**
```
개인정보보호법, GDPR 등
"고객 데이터가 어디에 저장되나요?"
→ 대답할 수 없음 = 법적 문제
```

:::callout[warning]
노코드 도구 자체가 나쁜 게 아니에요!
하지만 회사 데이터를 연결할 때는 반드시 보안팀과 상의해야 해요.
:::

### 안전하게 사용하려면?

**회사 차원에서:**
1. 승인된 도구 목록 관리
2. SSO(싱글사인온) 연동
3. 정기적인 접근 권한 감사
4. 퇴사자 계정 즉시 비활성화

**개인 차원에서:**
1. 회사 데이터 연결 전 보안팀 문의
2. 개인 계정 말고 회사 계정 사용
3. 공유 설정 꼼꼼히 확인
4. 민감 정보는 노코드 도구에 넣지 않기

:::conversation
