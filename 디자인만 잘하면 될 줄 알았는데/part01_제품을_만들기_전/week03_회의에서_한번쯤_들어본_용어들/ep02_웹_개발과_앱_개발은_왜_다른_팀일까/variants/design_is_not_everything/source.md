# 웹개발과 앱개발의 차이 (완성)

## 웹 개발

**기술 스택**: HTML, CSS, JavaScript, React 등
**장점**: 
- 크로스 플랫폼 (모든 기기에서 동작)
- 업데이트 용이
- 개발 비용 저렴

**단점**:
- 네이티브 기능 제한
- 성능 상대적으로 느림

## 앱 개발

**기술 스택**: 
- iOS: Swift, Objective-C
- Android: Kotlin, Java

**장점**:
- 높은 성능
- 네이티브 기능 활용
- 오프라인 사용 가능

**단점**:
- 플랫폼별 개발 필요
- 스토어 심사 필요
- 높은 개발 비용

## 📝 Feedback

### 현재 잘된 점
- 핵심 개념을 명확하게 설명
- 실무 예시와 사례 제공
- 단계별 가이드 포함

### 개선이 필요한 부분

1. **디자이너 관점 추가**
   - 디자이너가 이해해야 할 핵심 포인트
   - 디자인 작업과의 연관성
   - 협업 시 주의사항

2. **실무 대화 예시**
   - 팀 미팅에서의 실제 대화
   - 갈등 상황과 해결 방법
   - 효과적인 커뮤니케이션 팁

3. **실패 사례와 교훈**
   - 실제 프로젝트 실패 경험
   - 피해야 할 함정들
   - 배운 교훈과 개선 방안

4. **실용적 도구/템플릿**
   - 바로 사용 가능한 템플릿
   - 추천 도구와 서비스
   - 체크리스트와 가이드라인

5. **시각적 요소 강화**
   - 프로세스 다이어그램
   - 비교 표와 차트
   - 인포그래픽 추가 제안
