# 웹과 앱의 UX - 핵심 요약 카드뉴스 버전

<!-- variant: v3, type: 핵심 요약 카드뉴스 버전 -->

## 카드 1: 표지

**웹과 앱, 뭐가 다를까?**

같은 서비스라도 플랫폼마다 UX가 달라야 하는 이유

---

## 카드 2: 핵심 차이

| 웹 | 앱 |
|----|----|
| 🖱️ 마우스/키보드 | 👆 터치/제스처 |
| 🔗 URL로 바로 접근 | 📲 설치 후 사용 |
| ❌ 푸시 알림 제한 | ✅ 푸시 알림 가능 |
| 🔄 항상 최신 버전 | 📥 업데이트 필요 |

---

## 카드 3: 웹이 유리한 상황

✅ **웹을 선택하세요**

- SEO로 신규 유저 유입
- 가벼운 정보 탐색
- 공유/바이럴 중요
- 설치 없이 빠른 접근

---

## 카드 4: 앱이 유리한 상황

✅ **앱을 선택하세요**

- 자주 사용하는 서비스
- 카메라/GPS 등 기기 기능
- 푸시 알림으로 재방문 유도
- 오프라인 사용 필요

---

## 카드 5: 흔한 실수

❌ **이렇게 하지 마세요**

- 웹 디자인 그대로 앱에 적용
- 마우스 호버 효과를 앱에 사용
- 앱에서 새 탭으로 열기
- 플랫폼 가이드라인 무시

---

## 카드 6: 핵심 원칙

:::yama
**"플랫폼을 존중하라"**

웹은 웹답게, 앱은 앱답게.
사용자는 이미 각 플랫폼에 익숙합니다.
:::
