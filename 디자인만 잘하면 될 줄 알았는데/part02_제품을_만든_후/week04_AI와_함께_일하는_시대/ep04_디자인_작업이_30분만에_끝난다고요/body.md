# 디자인 작업이 30분만에 끝난다고요

## "이 작업 예전엔 하루 걸렸는데..."

같은 작업인데 요즘은 30분이면 끝나요. AI 도구들이 반복적인 작업을 대신해주거든요. 디자이너의 역할이 바뀌고 있어요.

## AI가 도와주는 디자인 작업

### 1. 이미지 생성

```
"파란 하늘 아래 카페에 앉아있는 사람 이미지 만들어줘"
→ 몇 초 만에 이미지 생성

활용:
- 목업용 이미지
- 프레젠테이션 비주얼
- SNS 콘텐츠
```

### 2. 배경 제거/변경

```
예전: 누끼 따는 데 30분
지금: AI가 자동으로 1초

활용:
- 상품 이미지
- 배너 제작
- 합성 작업
```

### 3. 텍스트 디자인

```
"이 문구를 인스타그램 카드뉴스에 맞게 레이아웃해줘"
→ 여러 시안 제안

활용:
- SNS 콘텐츠
- 배너
- 광고 소재
```

### 4. UI 생성

```
"로그인 화면 디자인해줘. 미니멀하게."
→ Figma 플러그인으로 UI 생성

활용:
- 초기 아이디어 탐색
- 와이어프레임
- 프로토타입
```

## 실제 도구들

### 이미지 생성

```
Midjourney: 고품질 아트워크
DALL-E: OpenAI의 이미지 생성
Stable Diffusion: 오픈소스, 커스터마이징 가능
Adobe Firefly: 상업적 사용 안전
```

### 디자인 보조

```
Figma AI: UI 생성, 자동 레이아웃
Canva Magic Design: 템플릿 기반 자동 디자인
Adobe Sensei: 편집 자동화
Remove.bg: 배경 제거
```

### 카피라이팅

```
"이 배너에 들어갈 카피 5개 뽑아줘"
→ AI가 여러 문구 제안
```

## AI로 빨라지는 작업들

### 반복 작업

```
예전:
- 배너 20개 사이즈별로 리사이징
- 일일이 수작업

지금:
- AI가 자동 리사이징
- 레이아웃도 자동 조정
```

### 에셋 생성

```
예전:
- 스톡 이미지 검색
- 라이선스 확인
- 색상 보정

지금:
- AI로 필요한 이미지 바로 생성
- 라이선스 걱정 없음 (도구에 따라 다름)
```

### 아이디어 탐색

```
예전:
- 레퍼런스 찾기
- 하나씩 시안 만들기

지금:
- AI에게 "5가지 다른 스타일로" 요청
- 빠르게 방향 결정
```

## 디자이너의 역할 변화

### AI가 대체하는 것

```
- 반복적인 작업
- 기초적인 에셋 생성
- 단순 편집 작업
```

### 사람이 해야 하는 것

```
- 문제 정의
- 사용자 이해
- 브랜드 가이드라인
- 최종 품질 판단
- 창의적 방향 설정
```

:::callout[important]
AI는 도구예요. 좋은 디자이너가 AI를 쓰면 더 좋은 결과물이 나와요.
나쁜 디자이너가 AI를 써도 마법처럼 좋아지진 않아요.
:::

## 주의할 점

### 1. 저작권 문제

```
AI 생성 이미지의 저작권은 아직 불명확해요.
상업적 사용 시 도구별 정책 확인 필요.

안전한 도구:
- Adobe Firefly (학습 데이터 검증됨)
- 라이선스 명확한 도구 선택
```

### 2. 브랜드 일관성

```
AI가 생성한 것은 브랜드와 안 맞을 수 있어요.
가이드라인에 맞게 수정 필요.
```

### 3. 품질 검수

```
AI 결과물 그대로 쓰지 마세요.
디테일 확인, 품질 검수는 필수.
```

## 효율적인 워크플로우

```
1. 문제 정의 (사람)
2. 방향 설정 (사람)
3. 초안 생성 (AI)
4. 검토 & 선택 (사람)
5. 상세 보완 (사람 + AI)
6. 최종 품질 검수 (사람)
```

## 정리

:::callout[important]
AI는 반복 작업을 빠르게 해줘요.
절약한 시간으로 더 중요한 문제에 집중하세요.
:::

**핵심:**
- AI가 반복 작업을 대신해줘요
- 이미지 생성, 편집, 리사이징 등이 빨라졌어요
- 디자이너의 역할은 "판단"으로 이동해요
- 저작권과 브랜드 일관성에 주의하세요
- AI 결과물은 반드시 검수하세요
