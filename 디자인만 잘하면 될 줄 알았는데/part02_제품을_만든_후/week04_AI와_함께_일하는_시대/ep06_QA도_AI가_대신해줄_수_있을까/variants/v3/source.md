## 카드 2: AI가 잘하는 QA

✅ **AI에게 맡기세요**

- 📝 테스트 케이스 생성
- 🔄 회귀 테스트 자동화
- 📸 스크린샷/UI 비교
- 📊 버그 패턴 분석
- ⚡ 대량 반복 테스트

---

## 카드 3: AI가 못하는 QA

❌ **사람이 해야 해요**

- 👤 사용자 경험 평가
- 🎯 직관적 사용성 판단
- 🏢 비즈니스 맥락 이해
- 🔍 창의적 탐색 테스트
- ♿ 접근성 테스트

---

## 카드 4: AI QA 도구들

🛠️ **대표 도구**

| 용도 | 도구 |
|------|------|
| 테스트 자동화 | Testim, Mabl |
| 시각적 테스트 | Applitools, Percy |
| 성능 테스트 | LoadRunner AI |
| 보안 스캔 | Snyk, Checkmarx |

---

## 카드 5: 효율적인 분업

🤝 **AI + 사람 = 최강**

| AI (양) | 사람 (질) |
|---------|----------|
| 대량 테스트 | UX 판단 |
| 반복 작업 | 탐색 테스트 |
| 데이터 분석 | 최종 검증 |
| 패턴 감지 | 맥락 이해 |
