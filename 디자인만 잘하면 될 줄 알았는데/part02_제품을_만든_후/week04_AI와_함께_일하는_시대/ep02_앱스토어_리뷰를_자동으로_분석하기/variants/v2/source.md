# 앱스토어 리뷰를 자동으로 분석하기 - 실무 체크리스트 버전

<!-- variant: v2, type: 실무 체크리스트 버전 -->

## 핵심 요약

앱스토어 리뷰를 AI로 분석하면 수천 개의 리뷰에서 패턴과 인사이트를 빠르게 추출할 수 있다. 정기적인 분석으로 사용자 목소리를 제품에 반영하자.

---

## 리뷰 분석 프로세스

| 단계 | 설명 | 도구 |
|------|------|------|
| 수집 | 리뷰 데이터 추출 | AppFollow, Sensor Tower, 직접 크롤링 |
| 정제 | 노이즈 제거, 분류 | 스프레드시트, Python |
| 분석 | AI로 패턴 분석 | ChatGPT, Claude |
| 인사이트 | 액션 아이템 도출 | 팀 논의 |
| 적용 | 로드맵 반영 | Jira, Linear |

## AI 분석 프롬프트 템플릿

### 감정 분석
```
아래 앱스토어 리뷰들을 분석해줘:
1. 긍정/부정/중립 비율
2. 긍정적 키워드 Top 5
3. 부정적 키워드 Top 5
4. 전체 감정 요약

[리뷰 데이터]
```

### 주제별 분류
```
아래 리뷰들을 주제별로 분류해줘:
- 기능 요청
- 버그 리포트
- UX/UI 피드백
- 성능 이슈
- 칭찬
- 기타

각 주제별 대표 리뷰 3개씩 선정해줘.

[리뷰 데이터]
```

### 인사이트 도출
```
아래 리뷰 분석 결과를 바탕으로:
1. 가장 시급한 개선 사항 3가지
2. 사용자가 가장 좋아하는 기능 3가지
3. 경쟁사 대비 강점/약점
4. 다음 업데이트에 반영할 제안

[분석 결과]
```

### 우선순위 분류
| 카테고리 | 기준 | 조치 |

| P0 | 크래시, 데이터 손실 | 즉시 수정 |

## 실무 팁

:::callout[tip]
**"별점 1점" 리뷰가 보물**

왜?
- 진짜 문제점이 드러남
- 개선하면 충성 고객 전환 가능
- 다른 유저도 같은 불만일 수 있음

→ 불만 리뷰를 피하지 말고 파헤치세요
:::

:::callout[warning]
**가짜 리뷰에 주의**

의심 신호:
- 같은 날 대량 등록
- 비슷한 문체
- 과하게 긍정/부정
- 앱 기능과 무관한 내용

→ 노이즈 제거 후 분석하세요
:::
