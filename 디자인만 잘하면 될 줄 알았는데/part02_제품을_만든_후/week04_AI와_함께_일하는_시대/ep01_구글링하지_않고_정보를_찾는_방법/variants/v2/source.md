# 구글링하지 않고 정보를 찾는 방법 - 실무 체크리스트 버전

<!-- variant: v2, type: 실무 체크리스트 버전 -->

## 핵심 요약

AI 검색(ChatGPT, Claude 등)은 정보를 빠르게 종합하고 맥락에 맞는 답변을 얻는 데 효과적이다. 구글 검색과 AI 검색의 장단점을 이해하고 상황에 맞게 활용해야 한다.

---

## 검색 방식 비교

| 비교 | 구글 검색 | AI 검색 |
|------|----------|--------|
| 결과 형태 | 링크 목록 | 종합된 답변 |
| 출처 확인 | 원본 링크 제공 | 출처 불분명 (일부 제공) |
| 최신성 | 실시간 업데이트 | 학습 데이터 기준 |
| 복잡한 질문 | 검색어 조합 필요 | 자연어로 질문 가능 |
| 후속 질문 | 새로 검색 필요 | 대화 이어가기 가능 |

---

## AI 검색 활용 체크리스트

### AI 검색이 적합한 경우
- [ ] 개념 설명이 필요할 때
- [ ] 여러 정보를 종합해야 할 때
- [ ] 후속 질문이 필요할 때
- [ ] 요약이 필요할 때
- [ ] 비교 분석이 필요할 때

### 구글 검색이 적합한 경우
- [ ] 최신 뉴스/정보가 필요할 때
- [ ] 공식 문서/사이트가 필요할 때
- [ ] 특정 사이트 내 검색
- [ ] 정확한 출처 확인이 필수일 때
- [ ] 이미지/영상 검색

## 좋은 AI 질문 작성 체크리스트

### 질문 구조
- [ ] 역할 부여: "당신은 ~입니다"
- [ ] 맥락 제공: 상황 설명
- [ ] 구체적 요청: 원하는 형태 명시
- [ ] 제약 조건: 길이, 형식 등
- [ ] 예시 제공: 원하는 결과 예시

### 좋은 질문 예시
```
[역할] 당신은 시니어 PM입니다.
[맥락] 이커머스 앱의 장바구니 기능을 개선하려고 합니다.
[요청] 사용자 이탈을 줄이기 위한 개선 아이디어 5가지를 제안해주세요.
[형식] 각 아이디어에 대해 문제점, 해결책, 기대효과를 포함해주세요.
```

### 피해야 할 질문
- [ ] "~에 대해 알려줘" (너무 광범위)
- [ ] "좋은 거 추천해줘" (기준 불명확)
- [ ] "이거 해줘" (맥락 없음)

## AI 검색 결과 검증 체크리스트

### 필수 확인
- [ ] 사실 여부 (구글로 교차 확인)
- [ ] 출처 확인 가능한가?
- [ ] 최신 정보인가? (언제 기준?)
- [ ] 편향은 없는가?

### 검증이 특히 필요한 경우
- [ ] 통계/수치 정보
- [ ] 법률/규정 관련
- [ ] 의료/건강 정보
- [ ] 최신 이벤트/뉴스
- [ ] 인물 정보

## AI 도구별 특징

| 도구 | 특징 | 적합한 용도 |
|------|------|------------|
| ChatGPT | 범용, 플러그인 | 일반 질문, 글쓰기 |
| Claude | 긴 문서 처리 | 문서 분석, 요약 |
| Perplexity | 출처 명시 | 리서치, 팩트체크 |
| Copilot (Bing) | 실시간 검색 | 최신 정보 필요 시 |

## 프롬프트 템플릿

### 개념 설명 요청
```
"[용어]를 비개발자도 이해할 수 있게 설명해줘.
일상적인 비유를 사용하고, 3문장 이내로 요약해줘."
```

### 비교 분석 요청
```
"[A]와 [B]를 비교해줘.
표 형식으로 장단점, 적합한 상황을 정리해줘."
```

### 문서 초안 요청
```
"[주제]에 대한 [문서 종류] 초안을 작성해줘.
대상 독자는 [타겟]이고, 분량은 [분량]이야.
포함할 내용: [항목1], [항목2], [항목3]"
```

### 리뷰/피드백 요청
```
"아래 [문서/코드]를 리뷰해줘.
개선할 점과 좋은 점을 각각 3가지씩 알려줘.
[문서/코드 내용]"
```

## 실무 팁

:::callout[tip]
**"AI + 구글" 콤보가 최강**

1. AI로 먼저 개념 이해
2. 구글로 공식 문서/출처 확인
3. AI로 추가 질문/심화

→ 속도와 정확성 모두 잡기
:::

:::callout[warning]
**AI 답변을 그대로 쓰지 마세요**

주의할 점:
- 할루시네이션 (잘못된 정보)
- 저작권 문제
- 표절 이슈
- 맥락 오해

→ 항상 검토하고 수정하세요
:::
