## 카드 2: 구글 vs AI

🔍 **검색 방식의 차이**

| 구글 | AI |
|------|------|
| 🔗 링크 목록 | 💬 답변 제공 |
| 🕵️ 직접 탐색 | 📝 종합 정리 |
| ✅ 출처 명확 | ⚠️ 출처 확인 필요 |
| 📅 실시간 | 📚 학습 데이터 기준 |

---

## 카드 3: AI가 잘하는 것

✅ **AI에게 물어보세요**

- 개념 설명 & 요약
- 여러 정보 비교/종합
- 후속 질문 이어가기
- 복잡한 주제 정리
- 아이디어 브레인스토밍

---

## 카드 4: 구글이 필요할 때

🔍 **구글을 쓰세요**

- 📰 최신 뉴스/이벤트
- 📄 공식 문서/사이트
- ✅ 정확한 출처 필수
- 🖼️ 이미지/영상 검색
- 🔢 실시간 데이터

---

## 카드 5: 좋은 AI 질문법

💡 **이렇게 물어보세요**

```
❌ "마케팅 알려줘"

✅ "B2B SaaS 스타트업의
    초기 마케팅 전략 5가지를
    표 형식으로 장단점과 함께 정리해줘"
```

**맥락 + 형식 + 범위 = 좋은 답변**
