# 앱 심사에서 계속 거절당하는 이유

## "왜 자꾸 리젝되는 거죠?"

앱 개발 완료! 앱스토어에 올리기만 하면 되는 줄 알았는데... "심사 거절" 메일이 왔어요. 왜 그런 걸까요?

## 앱스토어 심사란?

앱스토어(Apple App Store, Google Play)는 아무 앱이나 올릴 수 없어요. ==모든 앱은 심사를 통과해야 사용자에게 공개됩니다.==

### 왜 심사를 할까?

- 사용자 보호 (악성 앱 차단)
- 플랫폼 품질 유지
- 정책 준수 확인

## 자주 거절당하는 이유들

### 1. 메타데이터 문제

**앱 이름/설명이 부정확**
```
❌ "최고의 앱" - 증명할 수 없는 주장
❌ 경쟁사 이름 언급
❌ 관련 없는 키워드 나열
```

**스크린샷 문제**
```
❌ 기기 프레임이 잘못됨
❌ 실제 앱과 다른 화면
❌ 저화질 이미지
```

### 2. 기능 문제

**앱이 제대로 작동 안 함**
```
❌ 크래시 발생
❌ 빈 화면
❌ 로그인 후 아무것도 안 됨
```

**기능이 너무 단순함 (Apple 자주 거절)**
```
❌ 웹사이트를 앱으로 감싼 것
❌ 단순한 웹뷰 앱
❌ 콘텐츠가 너무 적음
```

### 3. 개인정보 문제

:::callout[warning]
개인정보 관련은 아주 민감해요!
:::

**필수 누락 사항**
- 개인정보처리방침 링크
- 데이터 수집 동의
- 회원탈퇴 기능 (필수!)

### 4. 결제 문제

**인앱 결제 규정 위반**
```
❌ 외부 결제로 유도
❌ 앱 내에서 웹 결제 링크 제공
❌ 수수료 회피 시도
```

### 5. 콘텐츠 문제

**금지된 콘텐츠**
- 저작권 침해
- 불법 콘텐츠
- 특정 대상 비하
- 도박 (라이센스 없이)

## Apple vs Google 차이

| 항목 | Apple App Store | Google Play |
|------|-----------------|-------------|
| 심사 기간 | 1-3일 | 몇 시간-며칠 |
| 심사 엄격도 | 매우 엄격 | 상대적 유연 |
| 거절 시 대응 | 이의 제기 가능 | 수정 후 재제출 |
| 주요 거절 이유 | 기능/UX | 정책 위반 |

### Apple이 특히 까다로운 것

- 앱의 완성도
- 기본 기능 제대로 작동
- 웹뷰만 있는 앱 거절
- 결제 관련 정책

### Google이 주로 보는 것

- 악성코드/보안
- 개인정보처리방침
- 콘텐츠 가이드라인
- 광고 정책

## 디자이너가 확인해야 할 것

### 스크린샷 준비

**Apple App Store:**
- 6.5인치 (iPhone Pro Max)
- 5.5인치 (iPhone 8 Plus)
- iPad Pro 12.9인치

**Google Play:**
- 최소 2장, 최대 8장
- 최소 320px, 최대 3840px

### 앱 아이콘

- 모서리 둥근 처리 (Apple이 자동 적용)
- 투명 배경 금지 (Apple)
- 고해상도 필수

### 스크린샷 체크리스트

- [ ] 실제 앱 화면과 일치하는가?
- [ ] 기기 프레임이 정확한가?
- [ ] 해상도가 충분한가?
- [ ] 중요 기능이 잘 보이는가?
- [ ] 텍스트가 읽히는가?

## 거절됐을 때 대응법

### 1. 거절 사유 정확히 파악

심사 팀 메시지를 꼼꼼히 읽으세요. 어떤 가이드라인을 위반했는지 명시되어 있어요.

### 2. 문제 수정

해당 부분을 수정하고 다시 제출하세요.

### 3. 이의 제기 (필요시)

정당한 이유가 있다면 답변을 통해 설명할 수 있어요.

## 심사 전 체크리스트

:::callout[important]
심사 전 필수 확인 사항
:::

- [ ] 앱이 크래시 없이 작동하는가?
- [ ] 로그인/회원가입이 정상 작동하는가?
- [ ] 회원탈퇴 기능이 있는가?
- [ ] 개인정보처리방침이 있는가?
- [ ] 결제가 인앱결제로 되어있는가?
- [ ] 스크린샷이 실제 앱과 일치하는가?
- [ ] 테스트 계정 정보를 제공했는가?

## 정리

:::callout[important]
앱스토어 심사는 사용자를 보호하기 위한 과정이에요.
가이드라인을 미리 확인하고 준비하면 거절을 줄일 수 있어요.
:::

**핵심:**
- 심사 거절은 흔한 일, 당황하지 말기
- 거절 사유를 정확히 파악하고 수정
- 개인정보, 결제, 콘텐츠 정책 특히 주의
- 디자이너는 스크린샷, 아이콘 품질 체크
