# 왜 우리 서비스는 검색이 안 될까

## "구글에서 검색해도 안 나와요"

열심히 만든 서비스인데, 구글에서 검색해도 나오지 않아요. 왜 그럴까요?

## 검색엔진이 웹사이트를 찾는 방법

### 크롤링 → 인덱싱 → 랭킹

```
1. 크롤링: 구글 봇이 웹사이트를 방문해서 내용을 수집
2. 인덱싱: 수집한 내용을 구글 데이터베이스에 저장
3. 랭킹: 검색어에 맞는 결과를 순위대로 보여줌
```

==인덱싱이 안 되면 검색 결과에 나타나지 않아요.==

## 검색이 안 되는 이유들

### 1. 아직 인덱싱이 안 됐어요

새로 만든 사이트는 구글이 아직 방문하지 않았을 수 있어요.

**확인 방법:**
```
구글에서 검색: site:우리사이트.com
```
결과가 없으면 인덱싱이 안 된 것.

### 2. 크롤링을 막아놨어요

:::callout[warning]
개발 중에 검색엔진 차단 설정을 했다가 풀지 않는 경우가 많아요!
:::

**robots.txt에서 차단 중**
```
# 이렇게 되어있으면 전체 차단
User-agent: *
Disallow: /
```

**메타 태그로 차단 중**
```html
<meta name="robots" content="noindex, nofollow">
```

### 3. 콘텐츠가 부족해요

- 페이지 내용이 너무 적음
- 중요한 정보가 이미지로만 되어있음
- 다른 사이트와 중복되는 콘텐츠

### 4. 기술적 문제

**JavaScript만으로 렌더링**
- 구글 봇이 JavaScript를 완벽히 실행하지 못할 수 있음
- 서버 사이드 렌더링(SSR) 필요

**로딩이 너무 느림**
- 구글은 속도도 평가 요소로 봄

**모바일 친화적이지 않음**
- 반응형이 아닌 사이트는 불리

## SEO 기본 체크리스트

### 기술적 SEO

- [ ] robots.txt에서 차단 안 됐는지
- [ ] sitemap.xml이 있는지
- [ ] HTTPS 적용됐는지
- [ ] 모바일 친화적인지
- [ ] 로딩 속도가 적당한지

### 콘텐츠 SEO

- [ ] 페이지 제목(title)이 적절한지
- [ ] 메타 설명(description)이 있는지
- [ ] 제목 태그(h1, h2)를 잘 사용하는지
- [ ] 이미지에 alt 텍스트가 있는지
- [ ] 콘텐츠가 충분한지

## 디자이너가 신경 써야 할 것

### 1. 텍스트는 텍스트로

```
❌ 중요한 정보를 이미지로 만들기
✅ 텍스트로 제공하고 CSS로 스타일링
```

검색엔진은 이미지 안의 글자를 읽기 어려워요.

### 2. 이미지 alt 텍스트

```html
❌ <img src="product.jpg">
✅ <img src="product.jpg" alt="파란색 스니커즈 정면">
```

이미지에 alt 텍스트가 있으면 이미지 검색에도 노출돼요.

### 3. 의미 있는 URL

```
❌ /page?id=123
✅ /products/blue-sneakers
```

### 4. 페이지 제목 구조

```
❌ 모든 페이지가 같은 제목
✅ 각 페이지마다 고유하고 설명적인 제목
```

## 빠르게 인덱싱되게 하려면

### Google Search Console 사용

1. Search Console에 사이트 등록
2. sitemap.xml 제출
3. URL 검사 도구로 개별 페이지 요청

### 외부 링크 확보

다른 사이트에서 우리 사이트로 링크가 있으면 크롤링이 빨라져요.

## 검색 순위를 높이려면

:::callout[tip]
검색 첫 페이지에 나오려면 시간과 노력이 필요해요.
:::

**중요 요소:**
1. 콘텐츠 품질
2. 사용자 경험
3. 백링크 (외부 사이트에서 오는 링크)
4. 사이트 속도
5. 모바일 친화성

## 정리

:::callout[important]
검색에 나오려면 먼저 인덱싱이 되어야 해요.
robots.txt, sitemap, 콘텐츠 품질을 확인하세요.
:::

**핵심:**
- site:도메인.com 으로 인덱싱 여부 확인
- robots.txt가 크롤링을 막고 있지 않은지 확인
- 텍스트 콘텐츠가 충분해야 함
- Search Console로 인덱싱 요청 가능
- SEO는 단기간에 되는 게 아님
