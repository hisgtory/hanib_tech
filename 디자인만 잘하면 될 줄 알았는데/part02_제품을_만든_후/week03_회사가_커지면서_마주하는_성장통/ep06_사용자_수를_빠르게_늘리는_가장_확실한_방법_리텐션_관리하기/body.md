# 사용자 수를 빠르게 늘리는 가장 확실한 방법, 리텐션 관리하기

## "유저가 안 늘어요"

마케팅 비용을 계속 쓰는데 유저 수가 안 늘어요. 새로 들어오는 만큼 나가는 것 같아요. 뭐가 문제일까요?

## 리텐션이 성장의 핵심

### 밑 빠진 독에 물 붓기

```
이번 달: 1,000명 가입, 900명 이탈 → 순증 100명
다음 달: 1,000명 가입, 900명 이탈 → 순증 100명

마케팅 비용은 계속 쓰는데 성장은 느림
```

:::callout[warning]
새 유저를 데려오는 것보다 기존 유저를 유지하는 게 훨씬 싸요.
:::

### 리텐션이 좋으면

```
이번 달: 1,000명 가입, 300명 이탈 → 순증 700명
다음 달: 1,000명 가입, 300명 이탈 → 순증 700명 + 기존 700명 유지

→ 복리로 쌓여감
```

==리텐션은 복리처럼 작동해요. 작은 차이가 큰 격차를 만들어요.==

## 리텐션 측정하기

### 기본 리텐션 지표

```
Day 1 리텐션: 가입 다음 날 접속 비율
Day 7 리텐션: 가입 7일 후 접속 비율
Day 30 리텐션: 가입 30일 후 접속 비율
```

### 리텐션 커브 해석

```
좋은 리텐션 커브:
Day 1: 40% → Day 7: 25% → Day 30: 15% (안정적으로 유지)

나쁜 리텐션 커브:
Day 1: 30% → Day 7: 5% → Day 30: 1% (계속 떨어짐)
```

:::callout[important]
리텐션 커브가 어느 시점에서 수평이 되어야 해요.
계속 떨어지면 결국 0이 돼요.
:::

### 코호트 분석

```
코호트: 같은 시기에 가입한 사용자 그룹

1월 가입자 리텐션 vs 2월 가입자 리텐션

→ 개선이 되고 있는지 확인 가능
```

## 리텐션 개선 전략

### 1. 첫 경험 개선 (Day 1)

```
첫날 이탈이 가장 많음
→ 온보딩 개선
→ 핵심 가치 빠르게 전달
```

### 2. Aha moment 도달

```
"아, 이거 괜찮네!" 순간을 찾기

예:
- 페이스북: 7일 내 10명 친구 추가
- 슬랙: 2,000개 메시지 전송
- 줌: 첫 화상회의 완료
```

:::callout[tip]
리텐션 높은 사용자들의 공통 행동을 찾으세요.
그게 Aha moment 힌트예요.
:::

### 3. 습관 형성

```
일회성 사용 → 습관적 사용

도움이 되는 것:
- 알림/리마인더
- 연속 사용 보상
- 진전 가시화 (스트릭, 레벨 등)
```

### 4. 재방문 유도

```
이탈 위험 사용자 감지
→ 이메일, 푸시 알림
→ 돌아올 이유 제공 (새 기능, 혜택)
```

### 5. 제품 품질 개선

```
버그 많으면 → 떠남
느리면 → 떠남
UX 불편하면 → 떠남

기본기가 중요
```

## 이탈 원인 파악하기

### 1. 이탈자 설문

```
"왜 떠나셨어요?"

선택지:
- 필요한 기능이 없어서
- 다른 서비스가 더 좋아서
- 사용법이 어려워서
- 자주 안 쓰게 돼서
```

### 2. 행동 분석

```
이탈 전 마지막 행동은?
- 특정 기능에서 이탈?
- 에러 발생 후 이탈?
- 그냥 안 들어옴?
```

### 3. 비교 분석

```
리텐션 높은 사용자 vs 낮은 사용자
어떤 기능을 다르게 쓰나?
```

## 리텐션 vs 신규 획득

### ROI 비교

```
신규 유저 1명 획득 비용: 5,000원
기존 유저 1명 유지 비용: 500원

10배 차이!
```

### 집중 순서

```
1. 리텐션 먼저 개선
2. 리텐션이 안정되면 신규 획득에 투자
```

:::callout[important]
리텐션이 나쁜 상태에서 마케팅하면 돈 낭비예요.
:::

## 팀별 역할

### 제품/디자인

```
- 온보딩 개선
- UX 개선
- Aha moment 강화
```

### 개발

```
- 성능 개선
- 버그 수정
- 알림 시스템
```

### 마케팅

```
- 리마인더 캠페인
- 이탈 방지 이메일
- 재활성화 프로모션
```

### 데이터

```
- 리텐션 대시보드
- 코호트 분석
- 이탈 예측 모델
```

## 정리

:::callout[important]
성장의 핵심은 리텐션이에요.
새 유저를 데려오기 전에, 기존 유저가 머무는지 확인하세요.
:::

**핵심:**
- 리텐션이 나쁘면 아무리 마케팅해도 안 커요
- 리텐션은 복리로 작동해요
- Aha moment를 찾고 빨리 도달하게 하세요
- 습관을 형성하게 도와주세요
- 리텐션 먼저, 신규 획득은 그 다음
