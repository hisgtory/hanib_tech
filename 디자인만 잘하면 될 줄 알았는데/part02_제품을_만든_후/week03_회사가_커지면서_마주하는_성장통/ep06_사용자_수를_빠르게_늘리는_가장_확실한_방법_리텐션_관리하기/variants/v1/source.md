# 사용자 수를 빠르게 늘리는 가장 확실한 방법, 리텐션 관리하기

## 스토리텔링 버전

:::conversation
**N**: 한입님, 마케팅 비용을 엄청 쓰는데 유저 수가 안 늘어요. 새로 들어오는 만큼 나가는 것 같아요...

**한입**: 밑 빠진 독에 물 붓기를 하고 계시네요.

**N**: 네?

**한입**: 리텐션 먼저 잡아야 해요. 안 그러면 아무리 마케팅해도 소용없어요.
:::

### 밑 빠진 독에 물 붓기

:::conversation
**한입**: 계산해볼게요.

**리텐션 나쁠 때:**
- 이번 달: 1,000명 가입, 900명 이탈 → 순증 100명
- 다음 달: 1,000명 가입, 900명 이탈 → 순증 100명
- 마케팅 비용은 계속 쓰는데 성장은 느림

**리텐션 좋을 때:**
- 이번 달: 1,000명 가입, 300명 이탈 → 순증 700명
- 다음 달: 1,000명 가입, 300명 이탈 → 순증 700명 + 기존 700명 유지
- 복리로 쌓여감

**N**: 와, 차이가 엄청나네요!
:::

:::callout[important]
리텐션은 복리처럼 작동해요.
작은 차이가 시간이 지나면 큰 격차가 돼요!
:::

### 리텐션 측정하기

**기본 지표:**
```
Day 1 리텐션: 가입 다음 날 접속 비율
Day 7 리텐션: 가입 7일 후 접속 비율
Day 30 리텐션: 가입 30일 후 접속 비율
```

**좋은 리텐션 커브:**
```
Day 1: 40% → Day 7: 25% → Day 30: 15%
(어느 시점에서 수평이 됨)
```

**나쁜 리텐션 커브:**
```
Day 1: 30% → Day 7: 5% → Day 30: 1%
(계속 떨어져서 결국 0에 수렴)
```

### 리텐션 개선 전략

:::conversation
**N**: 그럼 어떻게 개선해요?

**한입**: 단계별로 접근해야 해요.

**1. 첫날 이탈 줄이기 (Day 1)**
- 온보딩 개선
- 핵심 가치 빠르게 전달
- 첫날이 가장 많이 빠져요!

**2. Aha Moment 도달시키기**
- 페이스북: 7일 내 친구 10명
- 슬랙: 메시지 2,000개
- "아, 이거 괜찮네!" 순간을 빨리 경험하게

**3. 습관 형성 돕기**
- 리마인더 알림
- 연속 사용 보상 (스트릭)
- 진전 보여주기 (레벨, 진행률)

**4. 이탈 위험군 잡기**
- 접속 빈도 떨어지는 사람 감지
- 돌아올 이유 제공 (새 기능, 혜택)
:::

### ROI로 보는 리텐션의 힘

:::conversation
**한입**: 비용으로 비교해볼게요.

**신규 유저 1명 획득 비용:** 5,000원
**기존 유저 1명 유지 비용:** 500원

10배 차이예요!

**N**: 그럼 기존 유저 유지가 훨씬 효율적이네요?

**한입**: 맞아요. 그래서 순서가 중요해요.

1. 리텐션 먼저 개선
2. 리텐션이 안정되면 신규 획득에 투자
:::

:::callout[warning]
리텐션이 나쁜 상태에서 마케팅하면 돈 낭비예요!
독부터 고치고 물을 부어야 해요.
:::

### 이탈 원인 파악하기

**1. 이탈자에게 물어보기**
```
"왜 안 쓰게 되셨어요?"
- 필요한 기능이 없어서
- 다른 서비스가 더 좋아서
- 사용법이 어려워서
- 자주 안 쓰게 돼서
```

**2. 행동 데이터 분석**
```
이탈 전 마지막 행동은?
- 특정 기능에서 이탈?
- 에러 발생 후 이탈?
- 그냥 안 들어옴?
```

**3. 잔존 유저와 비교**
```
리텐션 높은 유저 vs 낮은 유저
어떤 기능을 다르게 쓰나?
```
