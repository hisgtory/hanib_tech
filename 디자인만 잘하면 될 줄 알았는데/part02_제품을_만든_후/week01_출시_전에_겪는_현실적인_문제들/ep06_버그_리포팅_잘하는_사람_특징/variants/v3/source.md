## 카드 3: 버그 리포트 필수 4요소

📝 **이것만 기억하세요**

1. **뭘 했는지** → 재현 단계
2. **뭐가 나왔는지** → 실제 결과
3. **뭘 기대했는지** → 예상 결과
4. **어디서 했는지** → 환경 정보
