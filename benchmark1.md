# 벤치마킹 도서 목차

### 아는 만큼 보이는 IT 지식

1. 이것만 알면 된다! IT 기초 지식
    1. 인터넷의 동작 원리
        1. 클라이언트와 서버의 개념
        2. 서버에서 데이터를 받아오는 이유
        3. 서버의 역할
        4. 프론트엔드와 백엔드 개발의 개념
        5. IP 주소와 도메인 주소의 이해
        6. 인터넷 통신 과정과 프로토콜
        7. 주요 HTTP 상태 코드 이해하기
    2. 인터넷에서 데이터를 안전하게 지키는 법
        1. **방화벽의 개념과 형태**
        2. **내부망과 프록시 서버**
        3. 암호화의 개념과 종류
        4. 암호화를 적용한 네트워크 기술
    3. 웹과 앱 이해하기
        1. 웹의 개념과 발전 과정
        2. 반응형 웹의 이해
        3. 웹 앱과 모바일 앱의 차이
    4. 데이터베이스 이해하기
        1. 데이터베이스의 개념과 특징
        2. 트랜잭션의 이해
        3. SQL 기초와 활용
        4. RDBMS와 NoSQL의 특징
2. 개발자와 소통하기
    1. 개발 분야별 프로그래밍 언어
        1. 파이썬으로 시작하는 AI 개발
        2. 웹 개발의 3요소: HTML, CSS, JS
        3. 서버 개발: 자바, 코틀린, PHP
        4. 앱 개발: iOS와 안드로이드
        5. 크로스 플랫폼 개발의 이해
    2. 알아두면 좋은 개발 용어
        1. 라이브러리와 프레임워크
        2. SDK의 개념과 활용
        3. API 이해하기
        4. 데이터 포맷: JSON과 XML
    3. 안 된다고 하는 개발자의 속사정
        1. 담당자별 업무 범위
        2. 개발 거절의 숨은 이유
        3. 개발 가능성 파악하기
    4. 개발자와 잘 소통하는 법
        1. 요구사항 문서화하기
        2. 모호한 표현 피하기
        3. 대안 제시하는 법
        4. 문서 버전 관리하기
        5. 데이터 중심 대화법
